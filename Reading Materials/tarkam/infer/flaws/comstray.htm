Examples

Let's look at some more examples of common straying

'Elephant has tusk because it is a mammal'

A bull, which is a mammal doesn't have tusk.

'An aeroplane uses aviation fuel because it can fly'

All birds can fly. But they don't use aviation fuel.

In all the above examples, the reason is present in an instance, in which the thing to be proved is absent.
