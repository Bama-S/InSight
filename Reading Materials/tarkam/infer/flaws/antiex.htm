Examples

Let's see some more examples of antithetical reasoning

  • 'Computers can think, because they can perform calculations, like humans'

As such this argument is correct. But consider the argument,

'Computers can't think, because they are inanimate objects, like a pot'

This argument also seems to be correct, but it counters the first argument. Hence both suffer from antithetical  reasoning.

 

  • 'Mercury is shapeless, because it is a liquid like water'

   'Mercury has shape, because it is a metal like gold'

  The above two arguments counter each other. Also they don't suffer from any other flaw. Hence they are antithetical.
