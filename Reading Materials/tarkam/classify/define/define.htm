Now that we've completed classifications and have done some examples in definition, you should be able to refine these definitions using the classification hierarchy that you've learnt.

When we try to refine such definitions , first we describe the primary characteristics that it possesses. If such characteristics are common to more than one object we may go in for a refined definition.  The reason for refining is that, at every level of definition, we may narrow down to describe the object only, and nothing but the object . Thus a precise definition should be both sufficient and complete ,i.e, it shouldn't cover something outside  its domain nor should it leave out uncovered something in its domain.  This gives rise to the concept of Avyaapti  (leaving out that which should be included) and Ativyaapti ( including that which should be left out).

Lets put down the steps involved in refining a definition.

Steps Involved:

  1. First enumerate the trivial characteristics of the object under consideration.
  2. Using these characteristics, give a basic definition.
  3. This definition would invariably cover a much wider set of unrelated objects or
    may fail to cover even the set of related objects.
  4. Identify those characteristics that distinguish the object from this set of unrelated objects.
  5. Also find out the commonality in the set of related objects that it earlier failed to cover.
  6. Now refine the definition with the characteristics identified.
  7. As more adjectives are added the definition narrows down on the object or its group.
  8. If this definition isn't unique go back to step 4.

Now lets try to define a rose:

Step 1:
Identify the trivial characteristics :
Smell of a rose.

Step 2:
Define the rose:
Rose is made of earth(i.e. it comes from earth).
This justifies the presence of smell in it.

Step 3:
Identify the deficiency in the current definition:
The above definition isn't sufficient to define a rose. All things that are made of earth possess smell (ativyaapti).

Step 4:
Identify the distinguishing characteristics:
Rose is a flower that has good smell.

Step 5:
So the definition can be refined as follows:
Rose is a flower that is made of earth and possesses good smell.
But this is still insufficient, as all flowers posses good smell (ativyaapti).

Step 4:
Identify the distinguishing characteristics:
Rose has a distinct shape.

Step 5:
The final definition goes as follows:
Rose is a flower that is made of earth, possesses good smell, and is distinctly identified by its shape.

A rose is a rose is a rose.
