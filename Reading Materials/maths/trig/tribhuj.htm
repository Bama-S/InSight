Trigonometry

In trigonometry, we deal with the ratios of the sides of a triangle and each ratio is given a separate notation. Trigonometry, like geometry was used extensively in the construction of monuments, Vedis and the like. Aryabhatta, in fact, had derived the series notation for the sine of an angle.

In the right triangle ABC,
sin A = a / b ; sin C = c / b.
cos A = c / b ; cos C = a / b
tan A = a / c ; tan C = c / a.
Since B = 90, sin B = 1 ; cos B = 0 and tan B is infinity.


Tribhujank

Tribhujank is the term used to refer to a set of 3 numbers x, y, z such that x2 + y2 = z2. For example, the numbers 3, 4, 5 form a tribhujank as 32 + 42 = 52. The tribhujank is nothing but a Pythagorean triplet

From the tribhujank, it is possible to find out what the different trigonometric ratios are.
If  p,q,r is a tribhujank containing an angle A, then

sin A = q / r ; cos A = p / r ; tan A = q / p;
cosec A = r / q; sec A = r / p; cot A = p / q.

Addition and subtraction of Tribhujank

Let  (a, b, c)  and  (p, q, r) be two Tribhujank, containing angles A and B respectively.
Then, the tribhujank containing the angles A + B and A - B are obtained using the Urdhva-Tiryak sutra, which means vertically and crosswise. This is illustrated in the table below.

A a b c
B p q r
A+B ap - bq pb + aq cr
A-B ap + bq pb - aq cr

Any common factor in the result can be divided out. If the tribhujank of A is 4, 3, 5 then, the tribhujank of (90 - A) is 3, 4, 5.

Some examples :

(1) To find the Tribhujank of A + B & A - B where A is 4, 3, 5 and B is 12, 5, 13.

A 4 3 5
B 12 5 13
A+B (4*12)-(3*5)=33 (4*5)+(12*3)=56 5*13=65
A-B (4*12)+(3*5)=63 (12*3)-(4*5)=16 5*13=65

(2) To find the Tribhujank of 2A if that of A is 4, 3, 5

A 4 3 5
A 4 3 5
2A (4*4)-(3*3) = 7 (4*3)+(4*3)=24 5*5 = 25
A-A = 0 (4*4)+(3*3)=25
1		 (4*3)-(4*3)=0
0		 5*5 = 25
1

In general, the tribhujank of 2A is (x2 - y2), 2xy, z2
From the table, we also find that the tribhujank of 0 is 1, 0, 1.

(3) To find the tribhujank of A / 2 if that of A is 3, 4, 5

The tribhujank of A / 2 is found as follows. The middle element will remain the same. The 1st element is obtained by adding the 1st element of A and the last element of A. As before, any common factors are cancelled out. The third element is found from the other 2 elements.

Tribhujank of A is 3, 4, 5
Tribhujank of (A/2) is (3+5), 4, ?
We have to find what the value of '?' is. This reduces to 2, 1, ?, after cancelling the common factor, 2.
Thus, the third element, found from the other two is sqrt (5). In general, the tribhujank of A/2 is (x+z), y, ?. After cancelling the common factors, we find the third element from the other two.
