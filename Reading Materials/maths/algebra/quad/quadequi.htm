How is this equivalent to the formula we already know ?

The formula that we know is

x = - b ± sqrt ( b2 - 4ac )
                2 a

According to the Vedic method,

2 a x + b = ± sqrt ( b2 - 4ac )

Rearranging, we get x = - b ± sqrt ( b2 - 4ac )
                                                2 a

So, we find that this is not very different from what we already know.
