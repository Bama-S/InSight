Some Special Cases   

Case 1 :   Reciprocals

Example :
When the quadratic equation is of the form
    x + 1/x = 17/4
Splitting the right hand side into 4 + 1/4, we get
x + 1/x = 4 + 1/4
Hence, x = 4 or x = 1/4

It's just a matter of simple observation !

General rule : If the left-hand side is in the form of reciprocals, try to write the right hand side also as a sum of reciprocal numbers.

Some more examples :
1)   x + 1/x = 13/6
Here the right hand side can be split as 3/2 + 2/3
Hence, x = 3/2 or 2/3.
2)   x - 1/x = 5/6
Here the RHS can be written as 3/2 - 2/3 giving  x = 3/2 or x = -2/3

Case 2 : Sum of the two numerators is equal to the sum of the two denominators.

Example :
(3x+4)  = (5x+6)
(6x+7)    (2x +3)

Here 3x + 4 + 5x + 6 = 6x + 7 + 2x + 3 = 8x + 10
Therefore, equating sum of numerators, 8x + 10 to 0 gives x = -5/4
And equating difference of denominator & numerator, 6x + 7 - 3x - 4 to 0 gives 3x + 3 = 0 ; x = -1
 
General rule :
1)  Find the sum of the numerators. Equate it to 0 to get one value of x.
2)  Find the difference between between the numerator & denominator of the LHS or RHS. Equate it to 0 to get the other value of x.

More examples :
1)    (4x - 2) = (2x - 4)
       (3x + 1)    (3x - 7)

4x - 2 + 2x - 4 = 3x + 1 + 3x - 7 = 6x - 6.
6x - 6 = 0 gives x = 1.
(4x - 2) - (3x +1) = x - 3.
x - 3 = 0 gives x = 3.

2)    (15x + 22)  = (10x + 28)
      (12x + 17)   (13x + 33)

15x + 22 + 10x + 28 = 12x + 17 + 13x + 33 = 25x + 50.
25x + 50 = 0 gives x =  - 2.
(15x + 22) - (12x + 17) = 3x + 5.
3x + 5 = 0 gives x = - 5 / 3.

Case 3 : Special Type 3

Example :
  2                3               4               1     
------    +    ------    =   ------    +    -------   
(x+2)        (x+3)         (x+4)          (x+1) 

Do the following tests on the equation :

  • 2/1 + 3/1 = 4/1 + 1/1 (Denominator = coefficient of x) and
  • 2/2 + 3/3 = 4/4 + 1/1 (Here denominator = value of constant)

In all such cases one of the values of x = 0.
To find the other value of x,  x+2 + x+3 = 0 ; => 2x+5 = 0 giving x = -5/2

General rule :  For the equation
        a        +        b         =        c         +        d
    ---------        ---------          ---------          --------
    (ex + p)      (fx + q)         (gx + r)         (hx + s)

If
(i) (a / e) + (b / f)  = (c / g) + (d / h) and
(ii) (a / p) + (b / q) = (c / r) + (d / s)
then, x = 0.
For the other value of x, set either (ex + p) + (fx + q) = 0 (or) (gx + r) + (hx + s) = 0.

More examples :

            5         +        6            =         4        +         7
       ----------         ----------               ---------          ----------
       (2x + 10)      (2x + 12)            (2x + 8)        (2x + 14)

    (i) (5/2) + (6/2) = (4/2) + (7/2) and
    (ii) (5/10) + (6/12) = (4/8) + (7/14)
    Hence, x = 0.
    For the other value of x, (2x + 10) + (2x + 12)  = 0.
    4x + 22 = 0 giving  x = - 11/2.
