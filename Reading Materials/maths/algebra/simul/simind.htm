But what has Indian maths got to do with it ?

Indians have found a way to solve these equations. The Indian method offers a much simpler route to the solution, than the normal method that we are accustomed to.

                          

The Indian method is identical to the Cramer's rule for solving simultaneous equations, using determinants. If you're wondering what these determinants are, you'll have to be patient till you come across them in higher classes. If you really can't bear not to know what they are,then go ahead and do some research !!!

General method :

  • Consider the 2 equations
    a x + b y = c
    d x + e y = f
  • Find the numerator for x as Nx = (b * f) - (e * c)
  • Find the numerator for y as Ny = (d * c) - (a * f)
  • Find the denominator for both x & y as Dxy = (d * b) - (a * e)
  • x = Nx / Dxy  and  y = Ny / Dxy

More Examples

  • 3x + 2y = 29
    2x - y = 3

    Numerator for x = (2 * 3) - ((-1) * 29) = 6 + 29 = 35
    Numerator for y = (2 * 29) - (3 * 3) = 58 - 9 = 49
    Denominator for both x & y =  (2 * 2) - (3 * (- 1)) = 4 + 3 = 7

    x = 35 / 7 = 5
    y = 49 / 7 = 7

  • 10x - 15y = 35
    2x - 3y = 7

    Nx     = ( (- 15) * 7) - ((-3) * 35) = (- 105 + 105) = 0
    Ny = ( 2 * 35) - (10 * 7) = 70 - 70 = 0
    Dxy = (2 * (-15)) - (10 * (- 3)) = - 30 + 30 = 0

    Here, Nx = Ny = Dxy = 0
    When this happens, there are infinite solutions to the equations.

    In the above equation, note that (a / d) = (b / e) = (c / f)
    i.e. (10 / 2) = ( - 15 / -3) = (35 / 7) = 5.
